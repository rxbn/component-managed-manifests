= Parameters

The parent key for all of the following parameters is `managed_manifests`.

== `namespace`

[horizontal]
type:: string
default:: `syn-managed-manifests`

The namespace in which to deploy this component.


== Example

[source,yaml]
----
namespace: example-namespace
----

== `manifests`

[horizontal]
type:: object
default:: `{}`

Map of manifest files to render. Each key is rendered as a YAML file with the same name.

== `manifests.<filename>`

[horizontal]
type:: object
default:: `{}` (no manifests)

Each key under `managed_manifests.manifests` is rendered as a YAML file with the same name.

== Example

[source,yaml]
----
managed_manifests:
  manifests:
    01-mynamespace.yaml:
      apiVersion: v1
      kind: Namespace
      metadata:
        name: example
----
